### XML File 

XML-file which contains “Users” table dump. The structure of the file
```xml
<users>
    <user>
        <id>10</id>
        <name>User One</name>
        <email>user@mail.com</mail>
        <age>35</age>
    </user>
    <user>
    ....
    </user>
</users>
```

Usage
-----

```
php run.php 
```
